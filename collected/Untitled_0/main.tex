源似乎一直都是一个浪漫的人。不如说是懦弱？他自嘲地想。

当然，这并不是说他对待感情有多么热烈，不过他似乎确有一段明媚的时光，他努力地回想着那个女孩的脸，得到的却是一片雾霭。

他并不是一个现实的人，或许是基因无意的排列给了他这种天性？他依稀记得在初中的生物课上老师说过一些与此有关的话，不过无所谓，他当时正对着窗外那一枝海棠发呆——他从来就不是一个好学生，但那枝海棠倒是像极了他：孤单、无依无靠。源对学校一直没有什么强烈的情感，毕竟初中后他就没再上学。只记得因为学校建在了山窝里，午后的教室格外闷热。他总是在学校最差的班，虽然老师总是否认，但大家也都心知肚明。他总是坐在班级最靠窗的角落，观察窗外，也有时观察同学。课桌上总摊着一本日语教材——这是唯一能提起他对现实兴趣的一点宽慰了。他的初中教的是日语，他喜欢动漫。

不过他心里明白，他也曾经热爱过生活，热爱过现实，是现实，这个恶魔，一次又一次地击碎他的一切，又装作若无其事地离开。他不是天生就喜欢玩游戏，喜欢在赛博空间将自己的年华化作一串数据；他不是天生就沉默寡言，只是不愿再与外界有着更多无谓的联系；他不是天生就患上的抑郁，他还记得那段灰暗的日子，那是没有一点光的黑色，病魔压抑着他的精神\dots 他不愿赘述太多了，无边的荆棘就这样刺穿他的身体与灵魂，他看不到一点希望。

他的朋友，不，应该叫现实生活中唯一的好友，叶，似乎与他完全相反。当然了，叶不是什么品学兼优完美无瑕的爽文男主，只是凑巧还都过得去。叶似乎是一个很高能量的人，他们认识已经十年了，叶似乎总是在笑着的。在得知他抑郁后，叶第一次皱起了眉头，那是源从没见过的认真表情。在那之后，每个周末，叶都会冲到源的家，用他自己的话说“上蹿下跳”，用各种方式让源感受到哪怕一点现实的美好。也怪，两人的忙碌似乎在有一刻真的让源忘记了那些痛苦的过去，他第一次笑出了声。这份友情好像在不知不觉间变成了铠甲，虽然似乎一触即破，但也真切地守护着源。

什么？你问现在的源么？他啊，正站在医院的门口，身旁是他的挚友叶，那骨节分明的手里，紧攥的是病症痊愈的通知单。

慵懒的阳光似乎第一次照在他的身上。源忽然有一阵恍惚，他看向自己的双腿，抖抖自己的肩，就像刚刚身披铠甲踏过千万荆棘的勇者，在他面前，是无边的花海。

清风吹动他许久未剪的头发，或许，该剪个头了。在这一刻，源感觉自己如获新生。
