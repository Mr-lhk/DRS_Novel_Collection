# 小说汇总计划

[![License: CC BY-NC-ND 4.0](https://img.shields.io/badge/License-CC%20BY--NC--ND%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-nd/4.0/)

## 一、计划简介

近期校内同人文与原创小说创作风气旺盛，但许多优秀作品极易散失。为保存与展示这些珍贵的创作，本计划特设立 **公开作品收集与整理项目**，旨在：

- **避免同学们的创作被遗失**
- **为优秀作品提供更广泛的传播渠道**
- **通过整理与印刷，留下值得回顾的集体创作纪念**

本计划采用公开的 GitHub 仓库存放投稿内容，并将定期整理作品，印制少量实体合集。

本计划中，作品分为两类：

- **收录作品**：将进入文选，有机会被印刷。
- **非收录作品**：仅数字化整理与展示，不进入印刷版文选。

## 二、投稿须知与声明

为保障投稿人及相关权利人的合法权益，规范投稿、使用及展示行为，特制定如下须知与声明。投稿人向本计划提交作品即视为已充分阅读、理解并同意以下条款：

1. **授权方式及版权声明**  
   （1）除另有说明外，投稿人提交的作品默认以 **[CC BY-NC-ND 4.0](https://creativecommons.org/licenses/by-nc-nd/4.0/)** 协议授权本计划使用。你可以访问 [https://creativecommons.org/licenses/by-nc-nd/4.0/ ](https://creativecommons.org/licenses/by-nc-nd/4.0/ ) 获得一份该协议的副本。  
   （2）作品的著作权及其他相关权利由原作者享有。  
   （3）本计划在授权范围内对作品进行整理、编辑、展示、非商业性印刷及相关活动，不进行任何超出该授权条款的使用。  
   （4）如作品适用其他授权方式，将在作品源码及文章结尾处予以明确标注。

2. **署名方式**  
   投稿人可根据自身意愿选择使用实名、化名或不署名方式。本计划将按照投稿人选择的署名方式进行公开展示，上述选择不影响投稿人对作品权利的享有。

3. **涉及现实人物的授权义务**  
   若作品含有现实存在的姓名、肖像、特征或其他可识别其身份的内容（包括但不限于以其为明显原型的虚构角色），投稿人应保证已事先取得该人许可。因未取得许可而产生的全部法律责任由投稿人自行承担。

4. **个人信息保护**  
   投稿人如希望隐藏真实身份，或作品中涉及需保护的特定个人信息，应在投稿时明确说明。本计划管理员将在合理范围内协助进行化名处理或信息遮蔽，以保护相关人员隐私。

5. **合法性与内容规范要求**  
   （1）投稿内容须符合中华人民共和国现行法律、行政法规及公序良俗，不得含有违法、违规、侵权、人身攻击、侮辱、歧视等不当内容。  
   （2）投稿人应对其作品承担全部法律责任，包括因投稿行为而产生的任何纠纷或损害。

6. **条款更新**  
   本计划管理员有权根据实际情况对本须知与声明进行修订或更新，并在适当位置公布。更新后的条款自公布之时起生效，并适用于更新后所有新投稿内容。

## 五、问题反馈方式

由于项目仍处建设阶段，暂未设立工单或自动化处理系统。若你有任何疑问、建议或诉求，可通过以下任一方式反馈：

1. **直接联系管理员**  
   可以以邮件、微信、短信或当面沟通的方式表达你的问题与建议。

2. **在 GitHub 提交 Issue**  
   GitHub 仓库支持以 Issue 形式记录问题或建议。

目前人手有限，回复可能不够及时。若你有**紧急问题**，请直接与管理员面谈，并注明紧急性。
