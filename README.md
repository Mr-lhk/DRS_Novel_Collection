# 小说汇总计划

## 介绍

鉴于最近校内写同人文及小说风气的盛行，特开设此计划，旨在不让同学们辛苦创作的汗水付诸东流，也是为了更广泛地宣传精良的作品

本计划使用公开 Github 仓库存放投稿作品，并会定期印刷适量合集装订成册，向同学们免费发放

## 声明

本计划尊重各位同学的隐私，但请各位遵守以下规定，避免纠纷:

2. 基于作者个人意愿。部分作者可能会选择以不署名的形式投稿作品，在这种情况下，管理员可以视为拥有作者授权解决相关纠纷

3. 若同人文创作中提到了他人的名字或采用了现实原型，请向当事人取得许可后方可投稿

4. 若作者不愿透露自己的姓名或不愿透露文中拥有显示原型人物的真实身份，可在投稿时注明要求，由管理员代为将姓名改为化名，或隐藏作者身份

5. 提交内容需符合中华人民共和国法律及相关法规的内容，确保内容符合公序良俗，避免人身攻击，投稿人需对自己的稿件负责

6. 若管理员认为内容不合适，随时有权利拒绝投稿

7. 因统一排版需求，管理员在任何时刻都有权在不修改原作品内容的基础上更改作品的排版，或换用不同的排版方式，包括将手写或纸质作品数字化

8. 管理员有义务对投稿人明确表示不愿透露的信息保密

管理员随时有权利对此规定进行修改

## 反馈问题

我们目前尚未建成工单系统，所以若对本计划有任何建议或疑问，可通过以下方式反馈

1. 直接联系管理员，你可以通过邮件、微信、短信、当面告知等方式向管理员表达自己的疑问、诉求、建议等

2. 可以访问我们的[Github仓库](about:blank)，并以提交 Issues 的方式向我们反馈

由于人手不足，我们无法保证对于反馈的处理时间，若您有紧急需求，可以与管理员面谈，并点明紧急性
