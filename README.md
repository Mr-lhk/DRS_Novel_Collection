# 小说汇总计划

[![License: CC BY-NC-ND 4.0](https://img.shields.io/badge/License-CC%20BY--NC--ND%204.0-lightgrey.svg)](https://creativecommons.org/licenses/by-nc-nd/4.0/)

## 一、计划简介

近期校内同人文与原创小说创作风气旺盛，但许多优秀作品极易散失。为保存与展示这些珍贵的创作，本计划特设立 **公开作品收集与整理项目**，旨在：

- **避免同学们的创作被遗失**
- **为优秀作品提供更广泛的传播渠道**
- **通过整理与印刷，留下值得回顾的集体创作纪念**

本计划采用公开的 GitHub 仓库存放投稿内容，并将定期整理作品，印制少量实体合集。

本计划中，作品分为两类：

- **收录作品**：将进入文选，有机会被印刷。
- **非收录作品**：仅数字化整理与展示，不进入印刷版文选。

## 二、投稿须知与声明

为保障创作者与相关人员权益，请所有投稿者遵守以下规定：

1. **授权协议**  
   所有加入本计划的作品统一采用**[CC BY-NC-ND 4.0](https://creativecommons.org/licenses/by-nc-nd/4.0/)**协议授权。权利由原作者拥有，计划仅在协议范围内进行整理、展示与印刷。

2. **匿名投稿选项**  
   创作者可选择署名、化名或不署名。管理员会按投稿者要求处理。

3. **人物原型授权**  
   若同人文或小说中出现现实人物姓名或明显原型，请务必先取得当事人许可。

4. **身份保护**  
   若作者希望隐藏真实身份或文中涉及真实人物，请在投稿中说明，管理员将协助以化名替换或隐去相关信息。

5. **法律与公序良俗要求**  
   投稿内容须符合中华人民共和国法律法规，避免包含违法内容、人身攻击等不当内容。投稿者需对自己的稿件负责。

**管理员保留随时更新本说明文档的权利。**

## 五、问题反馈方式

由于项目仍处建设阶段，暂未设立工单或自动化处理系统。若你有任何疑问、建议或诉求，可通过以下任一方式反馈：

1. **直接联系管理员**  
   可以以邮件、微信、短信或当面沟通的方式表达你的问题与建议。

2. **在 GitHub 提交 Issue**  
   GitHub 仓库支持以 Issue 形式记录问题或建议。

目前人手有限，回复可能不够及时。若你有**紧急问题**，请直接与管理员面谈，并注明紧急性。
